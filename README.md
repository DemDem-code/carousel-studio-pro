# Carousel Studio Pro

A browser-based visual content creation tool for designing and exporting professional carousel slides built for content creators and social media strategists who need polished, on-brand visuals without relying on Canva or Figma.

## Why I Built This

I do content strategy and social media work for tech professionals, and I kept running into the same friction: existing tools were either too rigid for the kind of carousels I wanted to create, or required jumping between multiple apps. So I built my own.

Carousel Studio Pro started as a quick fix and evolved through multiple iterations into a full-featured slide design tool — complete with custom color palettes, typography controls, export sharpness optimization, and project save/load functionality.

## What It Does

- **Slide Design System** — Create intro slides, content slides (list-based), and CTA slides with full visual control
- **48-Color Palette** — Curated color system with support for solid backgrounds and gradients
- **Typography Controls** — 8 font families with granular size, weight, and spacing adjustments per slide element
- **Photo Backgrounds** — Upload images with automatic overlay generation for text readability
- **Slide Navigation Dots** — Auto-generated, position-aware dot indicators across all slides
- **Watermark System** — Configurable personal branding with size, weight, and opacity controls
- **High-Quality Export** — Renders at 3× resolution then downscales to 1080×1080 for sharp, platform-ready output
- **Project Save/Load** — Export your entire project as JSON and reload it later to continue editing

## How to Use

1. Clone or download this repo
2. Open `index.html` in any modern browser
3. Start designing — no install, no dependencies, no server required

## Tech Stack

- Vanilla HTML, CSS, and JavaScript (single-file architecture)
- [html2canvas](https://html2canvas.hertzen.com/) for high-fidelity slide rendering
- Google Fonts API for typography
